## WORKFLOW DE GENERACION

### Paso 1: Generar Imagenes
```bash
# Para cada prompt:
1. Copia el prompt completo
2. Pegalo en tu generador preferido (DALL·E 3, Midjourney, etc.)
3. Genera 2-3 variantes
4. Selecciona la mejor
```

### Paso 2: Descargar y Optimizar
```bash
# Convertir a WebP (Linux/Mac):
cwebp -q 85 imagen-original.jpg -o nombre-archivo-1200w.webp

# Crear version mobile (800x800):
cwebp -resize 800 800 -q 85 imagen-original.jpg -o nombre-archivo-800w.webp
```

### Paso 3: Subir al Servidor
```bash
# Estructura de carpetas:
/assets/images/social-proof/
  ├── reviews/
  │   ├── google-review-emergency-electrical-1200w.webp
  │   ├── google-review-electrical-installation-1200w.webp
  │   └── google-review-electrical-panel-1200w.webp
  ├── team/
  │   ├── team-electricians-group-culiacan-1200w.webp
  │   ├── senior-electrician-service-van-1200w.webp
  │   └── electrician-workshop-tools-1200w.webp
  └── before-after/
      ├── before-after-electrical-panel-repair-1200w.webp
      ├── before-after-outlet-installation-1200w.webp
      ├── before-after-led-lighting-1200w.webp

```

## CHECKLIST DE CALIDAD

Antes de publicar, verifica:

- [ ] **Tamano correcto:** 1200x800px para desktop
- [ ] **Formato WebP:** Compresion 80-85%
- [ ] **Alt text descriptivo:** SEO-optimizado
- [ ] **Lazy loading:** `loading="lazy"` en todas las imagenes
- [ ] **Responsive:** Version 800x800 para mobile
- [ ] **Consistencia visual:** Todas las imagenes tienen estilo similar
- [ ] **Sin watermarks:** Imagenes limpias sin marcas de agua
- [ ] **Texto legible:** En screenshots de Google Reviews
- [ ] **Realismo:** Fotos profesionales pero autenticas
- [ ] **Contexto local:** Ambiente mexicano/Culiacan visible

## GUIA DE ESTILO VISUAL

### Paleta de Colores (mantener consistencia):
- **Azul corporativo:** #1E40AF, #3B82F6
- **Amarillo acento:** #FCD34D (electricidad)
- **Verde exito:** #22c55e (CTAs, testimonios positivos)
- **Blanco limpio:** #ffffff (backgrounds)
- **Grises neutros:** #64748b, #1e293b (textos)

### Filtros Recomendados (Post-procesamiento):
- Temperatura: +5% (calido)
- Saturacion: -10% (profesional, no artificial)
- Contraste: +10% (claridad)

### Elementos a Evitar:
- ❌ Stock photos genericas (se nota)
- ❌ Iluminacion artificial/estudio (poco autentico)
- ❌ Modelos de publicidad (buscar realismo)
- ❌ Logos/textos visibles de competencia
- ❌ Fechas/timestamps futuros en reviews

---

## TIPS PARA MEJORES RESULTADOS

1. **Para Screenshots de Google**: Usa DALL·E 3, es el mejor en texto legible
2. **Para Fotos de Personas**: Midjourney v6 con `--style raw` da resultados mas realistas
3. **Para Before/After**: Asegura que el "after" sea visualmente mas atractivo
4. **Consistencia de Iluminacion**: Todas las fotos de equipo deben tener iluminacion similar
5. **Negative Prompts** (Stable Diffusion): "cartoon, illustration, 3d render, fake, staged, stock photo"
